# Engineer-Grade Agent Skills

This repository contains production-quality engineering skills for AI coding agents.

## Structure

- `skills/` — individual skill directories, each with a `SKILL.md`
- `commands/` — slash commands that compose skills into workflows
- `hooks/` — lifecycle hooks (pre/post tool call)
- `docs/` — guides and references
- `references/` — curated pattern libraries

## Install

```bash
/plugin marketplace add roanbrasil/engineer-grade-agent-skills
/plugin install engineer-grade-agent-skills
```
